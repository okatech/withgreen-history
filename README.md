# WithGreen History 🥗

サラダボウル専門店『**WithGreen**』創業者・**武文智洋（たけふみ ともひろ）社長の実際の発言だけ**で、創業から全国50店舗達成までを振り返るヒストリーWebアプリ。

## 🎯 概要

- 2016年5月、東京・神楽坂で1号店をオープンしたサラダボウル専門店『WithGreen』。
- 2026年3月14日、創業10年で全国50店舗を達成。
- 本サイトでは、武文社長が公開インタビュー・プレスリリースで実際に語った言葉だけを使い、その歩みをタイムラインと店舗数推移グラフで可視化しています。

## 🚀 デモ

GitHub Pages で公開中：  
👉 **https://okatech.github.io/withgreen-history/**

## 🛠 使用技術

- HTML / CSS / Vanilla JS（ビルド不要・単一ファイル）
- [Chart.js 4.x](https://www.chartjs.org/) （CDN経由）
- Google Fonts (Noto Sans JP)
